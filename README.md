# simple_monitoring_system
mqtt, sql,可視化ツールを利用した簡易モニタリングシステム

## 概要
各種IoTデバイス等のデータをmqttプロトコルで受け渡しを行って、そのデータをデータベース内に保存。
保存されたデータを可視化ツール上でグラフ化するもの。

### 使用ツール
- mqtt
  - mosquitto
  - paho-mqtt
- sql
  - MySQL
- 可視化ツール
  - grafana
